# Decrypter for Ockpo

Lenor

## Usage

### 快速解密模式

直接拖曳文件（或者用参数）打开，将进入快速解密模式。

这种模式下只接受文件，不接受文件夹。

具有五种解密模式：

1. 后缀名模式：将解密后的文件名后缀名改为 `.000`
2. 前缀名模式：在解密后的文件名前加上 `000-`
3. 前后缀名模式：同时使用前缀名和后缀名模式
4. 文件头伪装模式：将解密后的文件最前端添加特征文件头，伪装为 `Ockpo` 加密后的结果，便不会被二次加密。替换原本的文件
5. Raw 模式：不做任何处理，直接输出解密后的文件，并替换原本的文件（默认选项）
